== Description ==

This plug-in is designed specifically for publishing Big Time CrossFit's WODs to the public and our members through social media. It was created to simplify and automate communications of our daily workouts.

Note that this plug-in has not been tested on any other platform except for Big Time CrossFit's website and Facebook page. It will most likely need tweaking if you use it for your own purposes.

== Installation ==

Upload .zip file and activate plugin.
